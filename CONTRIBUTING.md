# Contributing to Awesome Neural Additive Models

Thank you for your interest in contributing!

## How to Contribute

1. Fork this repository
2. Add your resource to the appropriate section in `README.md`
3. Submit a pull request

## Guidelines

- Each paper entry should include: **title**, **authors**, **venue/year**, **one-line summary**, and links to **paper** and **code** (if available)
- Place entries in reverse chronological order within each section
- Verify that the paper is genuinely related to Neural Additive Models or neural GAMs
- Check for duplicates before adding
- Use the existing formatting style consistently
- One pull request per paper/resource when possible

## Entry Format

```markdown
- **Paper Title** — Authors — *Venue Year* — One-sentence summary. [[paper]](url) [[code]](url)
```

## What Belongs Here

- Papers that propose, extend, or analyze Neural Additive Models
- Papers that apply NAM or variants to specific domains
- Open-source implementations of NAM and variants
- Tutorials, blog posts, and educational resources about NAM
- Benchmark repositories that include NAM comparisons

## What Does Not Belong

- Papers that merely cite NAM without meaningful engagement
- Proprietary or closed-source tools
- Low-quality or predatory publications

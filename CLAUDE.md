# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

**Awesome Neural Additive Models** — a curated awesome-list of papers, implementations, and resources for Neural Additive Models (NAMs) and variants. This is a content-only repository (no build system or tests).

## Repository Structure

- `README.md` — The main awesome list (GitHub-facing), organized by: foundational work, architectural extensions, neural GAM alternatives, domain applications, theory, surveys, implementations, tutorials, and benchmarks
- `CONTRIBUTING.md` — Contribution guidelines for external contributors

## Editing Conventions

- Follow [awesome-list](https://github.com/sindresorhus/awesome) conventions
- Paper entry format: `- **Title** — Authors — *Venue Year* — Summary. [[paper]](url) [[code]](url)`
- Reverse chronological order within each section
- Use `##` for top-level categories, `###` for subcategories
- Always include arXiv/DOI links; include GitHub links when available
- Verify paper relevance to NAM before adding
